# Skill Swap Workspace 

This is the main workspace repository for **Skill Swap**, a platform designed for users to share and swap skills. This repository manages both the frontend and backend applications using Git Submodules.

---

## 📂 Project Structure

This workspace links two separate repositories:
*   **`client/`**: The frontend user interface (React/Next.js).
*   **`server/`**: The backend API and database integration (Node.js/Express/MongoDB).

---

## 🚀 How to Clone and Setup

If you or someone else wants to clone this entire workspace along with the submodules, run:

```bash
# Clone the parent repo along with all submodules
git clone --recurse-submodules [https://github.com/MahbubaSultanaEty/skill-swap-workspace.git](https://github.com/MahbubaSultanaEty/skill-swap-workspace.git)

# Move into the project directory
cd skill-swap-workspace
